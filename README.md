## Dataset

Dataset simulado generado directamente en Google Colab con información de productos, cantidades vendidas, precios y fechas de venta (enero a marzo 2026).

## Estructura del repositorio
analisis-ventas-tup/
├── datos/          # Dataset de ventas (generado al ejecutar el script)
├── scripts/        # Script de análisis en Python
├── resultados/     # Gráfico de evolución de ventas
├── README.md
└── .gitignore

## Cómo ejecutar

1. Abrir Google Colab
2. Clonar el repositorio
3. Ejecutar `scripts/analisis_ventas.py`
4. Los resultados se guardan en `/resultados`
